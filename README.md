# Football Analytics with Python

Python-based football analytics using Statsbomb Open Data and synthetic GPS benchmarks. 21 articles across 4 series, from raw event data to tactical analysis, physical performance, and long-form deep dives.

**Live site:** [twinanalytics.github.io/football-analytics-blog](https://twinanalytics.github.io/football-analytics-blog/)

---

## Series

| Series | Topic | Articles |
|---|---|---|
| 1 | Getting Started | Event data, pitch drawing, shot maps, pass networks, heatmaps |
| 2 | Tactical Analysis | xG, PPDA, through balls, freeze frames, carries, 360° data |
| 3 | Physical Performance | GPS metrics, sprint profiles, distance, accelerations, pressing space |
| 4 | Deep Dives | Messi, Barcelona 15/16, Women's World Cup, CL Finals, Leverkusen 2023/24 |

---

## Data Sources

**[Statsbomb Open Data](https://github.com/statsbomb/open-data)** — 3,464 matches, 75 seasons, event data + 360° tracking. Free for research and education.

**Synthetic GPS Dataset** — Physical performance data generated from peer-reviewed sports science benchmarks (Mohr et al. 2003, Bradley et al. 2009, Di Salvo et al. 2007).
